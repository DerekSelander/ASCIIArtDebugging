ASCIIArtDebugging

ASCIIArtDebugging is a fun yet useful way to debug UIImage & UIImageViews in order to see what gets passed around for the image at runtime. Instead of just looking at the memory address or properties while debugging, ASCIIArtDebugging takes the image and creates an ASCIIArt image in your console. You can control if the image is displayed in the object's description

NSLog("%@", imageView.image);

Or can enable the image to be only printed while debugging

po imageView.image
